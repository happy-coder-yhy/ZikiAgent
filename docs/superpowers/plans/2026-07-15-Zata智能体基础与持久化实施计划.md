- [ ] **Step 3: 实现确定性扫描器**

扫描器只生成以下不可泄密对象：

```python
@dataclass(frozen=True, slots=True)
class Finding:
    path: Path
    rule_id: str
    line_number: int

    def __repr__(self) -> str:
        return f"Finding(path={self.path!s}, rule_id={self.rule_id}, line={self.line_number})"
```

命令接口固定为：

```text
python scripts/secret_scan.py --tracked
python scripts/secret_scan.py --history
python scripts/secret_scan.py --paths path1 path2
```

规则至少覆盖 private key 头、Authorization/Bearer、非占位符 password/token 赋值和高风险平台凭据字段。`--tracked` 通过 `git ls-files -z` 获取文件；`--history` 使用 `git rev-list --objects --all` + `git cat-file --batch` 扫描 blob，只输出 commit/blob 对象 ID 与路径，绝不输出匹配行。发现任何命中返回退出码 1，扫描器异常返回 2。
